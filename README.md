# ?? Портфолио проектов по анализу данных

Добро пожаловать! В этом репозитории собраны мои аналитические проекты, демонстрирующие навыки работы с данными на языке Python. Каждый проект выполнен в формате Jupyter Notebook и включает полный цикл исследования: от предобработки данных до построения интерактивных графиков и формулирования бизнес-выводов.

---

## ?? Содержание портфолио

### ?? 1. Маркетинговая аналитика и A/B-тестирование
*   **[ab_test_knigi_i_sait.ipynb](./ab_test_knigi_i_sait.ipynb)** — Комплексный A/B-тест для оценки эффективности новой версии сайта книжного интернет-магазина и анализа изменения активности покупателей.
*   **[ab_test_user_activity.ipynb](./ab_test_user_activity.ipynb)** — Статистическое исследование пользовательской активности в развлекательном приложении после внедрения функции «бесконечной» ленты рекомендаций.

### ?? 2. Исследовательский анализ данных (EDA)
*   **[pd_games.ipynb](./pd_games.ipynb)** — Ретроспективный анализ мировой игровой индустрии, выявление факторов коммерческого успеха видеоигр и составление портрета геймера по регионам.
*   **[pd_np_plt_sns_phik_startapy.ipynb](./pd_np_plt_sns_phik_startapy.ipynb)** — Исследование исторических данных о финансировании стартапов, оценка выживаемости компаний и анализ инвестиционных раундов.
*   **[pd_plt_sns_phik_obshepit.ipynb](./pd_plt_sns_phik_obshepit.ipynb)** — Анализ рынка заведений общественного питания Москвы, изучение их географического распределения, вместимости и популярных форматов.
*   **[pd_plt_st_samokaty.ipynb](./pd_plt_st_samokaty.ipynb)** — Изучение поведения пользователей сервиса шеринга самокатов, проверка гипотез о затратах времени и расстоянии поездок.

## ?? Используемый технологический стек

В проектах активно применяются современные библиотеки экосистемы Python:

| Библиотека | Назначение в проектах |
| :--- | :--- |
| **pandas** | Первичный анализ, очистка данных, фильтрация, агрегация и трансформация |
| **
umpy** | Математические вычисления, векторизация и работа с многомерными массивами |
| **matplotlib** | Построение базовых графиков и кастомизация элементов визуализации |
| **seaborn** | Продвинутая статистическая визуализация (распределения, плотность, ящики с усами) |
| **scipy.stats** | Проверка статистических гипотез, проведение T-тестов и непараметрических тестов |
| **phik (?k)** | Расчет продвинутых коэффициентов корреляции для интервальных и категориальных признаков |

## ?? Общая структура каждого исследования

Каждая работа оформлена по стандартам индустрии разработки и содержит:
1. **Импорт модулей** — подключение библиотек и настройка отображения графиков.
2. **Загрузка и предобработка** — обработка пропущенных значений, аномалий, дубликатов и приведение типов данных.
3. **Исследовательский анализ (EDA)** — поиск скрытых взаимосвязей, закономерностей и трендов.
4. **Статистический анализ** — формулирование и проверка гипотез с помощью математических критериев.
5. **Выводы** — структурированные ответы на бизнес-задачи и рекомендации для принятия решений.
